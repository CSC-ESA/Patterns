SC-16(2): Transmission of Security and Privacy Attributes | Anti-spoofing Mechanisms ()
Implement anti-spoofing mechanisms to prevent adversaries from falsifying the security attributes indicating the successful application of the security process.
Control IdentifierSC-16(2)
Latest Sync Date19/12/24 09:18:14
DiscussionSome attack vectors operate by altering the security attributes of an information system to intentionally and maliciously implement an insufficient level of security within the system. The alteration of attributes leads organizations to believe that a greater number of security functions are in place and operational than have actually been implemented.
Related ControlsSI-3, SI-4, SI-7.
