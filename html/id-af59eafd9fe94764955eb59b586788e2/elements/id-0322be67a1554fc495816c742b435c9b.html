SA-3(1): System Development Life Cycle | Manage Preproduction Environment ()
Protect system preproduction environments commensurate with risk throughout the system development life cycle for the system, system component, or system service.
Control IdentifierSA-3(1)
Latest Sync Date19/12/24 09:18:14
DiscussionThe preproduction environment includes development, test, and integration environments. The program protection planning processes established by the Department of Defense are examples of managing the preproduction environment for defense contractors. Criticality analysis and the application of controls on developers also contribute to a more secure system development environment.
Related ControlsCM-2, CM-4, RA-3, RA-9, SA-4.
