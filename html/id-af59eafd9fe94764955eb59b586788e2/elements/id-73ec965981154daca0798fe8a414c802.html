SI-7(12): Software, Firmware, and Information Integrity | Integrity Verification ()
Require that the integrity of the following user-installed software be verified prior to execution: [Assignment: organization-defined user-installed software].
Control IdentifierSI-7(12)
Latest Sync Date19/12/24 09:18:14
DiscussionOrganizations verify the integrity of user-installed software prior to execution to reduce the likelihood of executing malicious code or programs that contains errors from unauthorized modifications. Organizations consider the practicality of approaches to verifying software integrity, including the availability of trustworthy checksums from software developers and vendors.
